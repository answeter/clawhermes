
# ClawHermes 快速启动指南

## 环境要求

- Python 3.12+
- DeepSeek API Key

## 启动步骤

### 1. 环境配置（首次使用）

在 `ClawHermes-main` 目录下运行：

```powershell
.\setup_env.ps1
```

这会将 `.env` 文件复制到正确位置。

### 2. 启动服务

你需要打开**两个**终端窗口：

#### 终端 1 - 启动 Gateway
```powershell
cd ClawHermes-main
uv run clawhermes gateway start
```

#### 终端 2 - 启动前端
```powershell
cd ClawHermes-main\web
python -m http.server 8000
```

### 3. 访问界面

在浏览器中打开：`http://localhost:8000`

## 使用说明

### 开始聊天

1. 在输入框中输入你的问题
2. 按 Enter 发送
3. 等待 AI 回复

### 功能特性

- **会话管理** - 左侧侧边栏保存历史对话
- **工具查看** - 点击"工具"按钮查看可用功能
- **设置配置** - 点击"设置"按钮调整模型和工具集

## 故障排除

### Gateway 无法启动

确保已运行 `setup_env.ps1`，并且 `.env` 文件中有有效的 API Key。

### 前端无法连接

确认两个服务都已启动，并且访问 `http://localhost:8000`（不是 127.0.0.1）。

## 文件说明

- `setup_env.ps1` - 配置环境脚本
- `start_all.ps1` - 显示启动说明
- `web/` - 前端界面目录
